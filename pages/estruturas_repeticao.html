6. Estruturas de Repetição (Loops)

Estruturas de repetição, ou loops, permitem executar um bloco de código múltiplas vezes, enquanto uma determinada condição for verdadeira ou por um número específico de iterações.

while

O loop while executa um bloco de código repetidamente enquanto uma condição especificada for verdadeira. A condição é verificada antes de cada execução do bloco.

Sintaxe:


while (condicao) {
    // Bloco de código a ser repetido
    // É importante que algo dentro do bloco altere a condição,
    // para evitar um loop infinito.
}

Exemplo: Contagem até 5


#include <stdio.h>

int main() {
    int contador = 1;

    while (contador <= 5) {
        printf("Contador: %d\n", contador);
        contador++; // Incrementa o contador para eventualmente tornar a condição falsa
    }
    printf("Fim do loop while.\n");
    return 0;
}

do...while

O loop do...while é semelhante ao while, mas com uma diferença crucial: o bloco de código é executado pelo menos uma vez, e a condição é verificada após a execução do bloco.

Sintaxe:


do {
    // Bloco de código a ser repetido
    // Algo aqui deve eventualmente tornar a condição falsa.
} while (condicao); // Note o ponto e vírgula aqui

Exemplo: Pedir entrada até ser válida


#include <stdio.h>

int main() {
    int numero;

    do {
        printf("Digite um numero positivo (ou 0 para sair): ");
        scanf("%d", &numero);

        if (numero < 0) {
            printf("Numero invalido! Tente novamente.\n");
        } else if (numero > 0) {
            printf("Voce digitou: %d\n", numero);
        }
    } while (numero < 0); // Repete se o número for negativo

    printf("Saindo do loop do...while.\n");
    return 0;
}

for

O loop for é ideal para quando se sabe de antemão quantas vezes um bloco de código deve ser repetido. Ele combina inicialização, condição e incremento (ou decremento) em uma única linha.

Sintaxe:


for (inicializacao; condicao; atualizacao) {
    // Bloco de código a ser repetido
}
  • Inicialização: Executada uma única vez, no início do loop. Geralmente usada para declarar e inicializar uma variável de controle.
  • Condição: Verificada antes de cada iteração. Se for verdadeira, o bloco é executado. Se falsa, o loop termina.
  • Atualização: Executada ao final de cada iteração. Geralmente usada para incrementar ou decrementar a variável de controle.

Exemplo: Tabuada do 5


#include <stdio.h>

int main() {
    int i;
    int numero = 5;

    printf("Tabuada do %d:\n", numero);
    for (i = 1; i <= 10; i++) {
        printf("%d x %d = %d\n", numero, i, numero * i);
    }
    printf("Fim do loop for.\n");
    return 0;
}

Também é comum o uso de break para sair de um loop prematuramente e continue para pular a iteração atual e ir para a próxima.

Exemplos de Códigos (Atividades Resolvidas)

Atividade 1: Soma dos Primeiros N Números Naturais

Leia um número inteiro N e calcule a soma de todos os números de 1 até N usando um loop for.


#include <stdio.h>

int main() {
    int n, i;
    int soma = 0;

    printf("Digite um numero inteiro positivo N: ");
    scanf("%d", &n);

    if (n < 1) {
        printf("N deve ser positivo.\n");
        return 1; // Indica erro
    }

    for (i = 1; i <= n; i++) {
        soma += i; // O mesmo que soma = soma + i;
    }

    printf("A soma dos numeros de 1 ate %d e: %d\n", n, soma);
    return 0;
}

Atividade 2: Contagem Regressiva com While

Faça um programa que realize uma contagem regressiva de 10 até 1, mostrando cada número na tela, e finalize com "FOGO!". Use while.


#include <stdio.h>

int main() {
    int contador = 10;

    while (contador >= 1) {
        printf("%d...\n", contador);
        contador--;
    }
    printf("FOGO!\n");
    return 0;
}

Desafio 1: Fatorial de um Número

Crie um programa que leia um número inteiro não negativo e calcule seu fatorial. O fatorial de N (N!) é $N \times (N-1) \times (N-2) \times \dots \times 1$. O fatorial de 0 é 1. Use o loop que achar mais adequado.

Desafio 2: Adivinhe o Número

Gere um número aleatório entre 1 e 100 (pesquise como usar `rand()` e `srand()` da biblioteca `stdlib.h`). Peça ao usuário para tentar adivinhar o número. A cada tentativa, informe se o palpite foi muito alto, muito baixo ou correto. Continue pedindo palpites até o usuário acertar. Conte o número de tentativas.

*
**
***
****
*****
Peça ao usuário o número de linhas desejado.
