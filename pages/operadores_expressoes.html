4. Operadores e Expressões

Operadores são símbolos especiais que realizam operações sobre operandos (valores ou variáveis). Uma expressão é uma combinação de operandos e operadores que resulta em um valor.

Operadores Aritméticos

Usados para realizar cálculos matemáticos.

  • + : Adição (Ex: a + b)
  • - : Subtração (Ex: a - b)
  • * : Multiplicação (Ex: a * b)
  • / : Divisão (Ex: a / b). Se ambos os operandos forem inteiros, o resultado é a parte inteira da divisão. Para obter um resultado com casas decimais, pelo menos um operando deve ser float ou double.
  • % : Módulo (Resto da divisão inteira. Ex: 10 % 3 resulta em 1)

Exemplo:


#include <stdio.h>

int main() {
    int a = 10, b = 4;
    float c = 10.0f, d = 4.0f;

    printf("a + b = %d\n", a + b);       // 14
    printf("a - b = %d\n", a - b);       // 6
    printf("a * b = %d\n", a * b);       // 40
    printf("a / b = %d\n", a / b);       // 2 (divisão inteira)
    printf("c / d = %.2f\n", c / d);   // 2.50 (divisão de ponto flutuante)
    printf("a %% b = %d\n", a % b);     // 2 (resto de 10/4)

    return 0;
}

Operadores Relacionais

Usados para comparar dois valores. O resultado de uma expressão relacional é um valor booleano (verdadeiro ou falso), que em C é representado por 1 (verdadeiro) ou 0 (falso).

  • == : Igual a (Ex: a == b)
  • != : Diferente de (Ex: a != b)
  • < : Menor que (Ex: a < b)
  • > : Maior que (Ex: a > b)
  • <= : Menor ou igual a (Ex: a <= b)
  • >= : Maior ou igual a (Ex: a >= b)

Exemplo:


#include <stdio.h>

int main() {
    int x = 5, y = 10;

    printf("x == y : %d\n", x == y);  // 0 (falso)
    printf("x != y : %d\n", x != y);  // 1 (verdadeiro)
    printf("x < y  : %d\n", x < y);   // 1 (verdadeiro)
    printf("x > y  : %d\n", x > y);   // 0 (falso)
    printf("x <= 5 : %d\n", x <= 5);  // 1 (verdadeiro)
    printf("y >= 10: %d\n", y >= 10); // 1 (verdadeiro)

    return 0;
}

Operadores Lógicos

Usados para combinar ou modificar expressões lógicas (booleanas).

  • && : E lógico (AND) - Verdadeiro somente se ambas as expressões forem verdadeiras. (Ex: cond1 && cond2)
  • || : OU lógico (OR) - Verdadeiro se pelo menos uma das expressões for verdadeira. (Ex: cond1 || cond2)
  • ! : NÃO lógico (NOT) - Inverte o valor lógico da expressão. (Ex: !cond1)

Exemplo:


#include <stdio.h>

int main() {
    int idade = 25;
    float salario = 3000.0f;
    _Bool temEmprego = 1; // Usando _Bool para clareza (ou int 0/1)

    // Exemplo 1: Maior de idade E tem emprego
    if (idade >= 18 && temEmprego) {
        printf("Maior de idade e empregado.\n");
    } else {
        printf("Nao atende aos criterios de maioridade e emprego.\n");
    }

    // Exemplo 2: Salário alto OU idade acima de 60
    if (salario > 5000.0f || idade > 60) {
        printf("Tem salario alto OU idade avancada.\n");
    }

    // Exemplo 3: NÃO é menor de idade
    if (!(idade < 18)) {
        printf("Nao e menor de idade.\n");
    }

    return 0;
}

Ordem de Precedência dos Operadores

A ordem de precedência define quais operadores são avaliados primeiro em uma expressão complexa. Parênteses () podem ser usados para alterar a ordem padrão de avaliação.

Ordem geral (do mais alto para o mais baixo, simplificado):

  1. Parênteses ()
  2. Operadores unários (!, ++, --, - unário)
  3. Multiplicativos (*, /, %)
  4. Aditivos (+, -)
  5. Relacionais (<, >, <=, >=)
  6. Igualdade (==, !=)
  7. E lógico (&&)
  8. OU lógico (||)
  9. Atribuição (=, +=, -=, etc.)

Exemplo: int resultado = 5 + 3 * 2; // resultado será 11 (3*2 primeiro, depois +5)

int resultado2 = (5 + 3) * 2; // resultado2 será 16 (5+3 primeiro, depois *2)

É uma boa prática usar parênteses para tornar expressões complexas mais claras, mesmo que a precedência padrão já forneça o resultado desejado.

Exemplos de Códigos (Atividades Resolvidas)

Atividade 1: Verificar se um número está em um intervalo

Leia um número inteiro e verifique se ele está entre 10 e 20 (inclusive).


#include <stdio.h>

int main() {
    int numero;
    printf("Digite um numero inteiro: ");
    scanf("%d", &numero);

    if (numero >= 10 && numero <= 20) {
        printf("O numero %d esta entre 10 e 20 (inclusive).\n", numero);
    } else {
        printf("O numero %d NAO esta entre 10 e 20.\n", numero);
    }
    return 0;
}

Atividade 2: Calculadora de IMC Simples

Leia o peso (float) e a altura (float) de uma pessoa. Calcule o IMC ($peso / (altura \times altura)$). Use operadores aritméticos.


#include <stdio.h>

int main() {
    float peso, altura, imc;

    printf("Digite seu peso em kg (ex: 70.5): ");
    scanf("%f", &peso);
    printf("Digite sua altura em metros (ex: 1.75): ");
    scanf("%f", &altura);

    if (altura > 0) { // Evita divisão por zero
        imc = peso / (altura * altura);
        printf("Seu IMC e: %.2f\n", imc);
    } else {
        printf("Altura invalida!\n");
    }
    return 0;
}
