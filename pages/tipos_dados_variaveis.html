3. Tipos de Dados e Variáveis

Tipos Primitivos

Em C, os tipos de dados primitivos são os blocos de construção básicos para armazenar informações. Os mais comuns são:

  • int: Usado para armazenar números inteiros (sem casas decimais). Ex: -10, 0, 100. Geralmente ocupa 2 ou 4 bytes.
  • float: Usado para armazenar números de ponto flutuante de precisão simples (com casas decimais). Ex: 3.14, -0.005, 2.7e5. Geralmente ocupa 4 bytes.
  • double: Usado para armazenar números de ponto flutuante de precisão dupla. Oferece mais precisão e uma faixa maior de valores que o float. Ex: 3.1415926535. Geralmente ocupa 8 bytes.
  • char: Usado para armazenar um único caractere. Ex: 'A', '7', '$'. Caracteres são, na verdade, armazenados como números (seus valores na tabela ASCII). Geralmente ocupa 1 byte.
  • void: Um tipo especial que significa "sem tipo" ou "nenhum valor". Usado principalmente para indicar que uma função não retorna valor ou não possui parâmetros.

Modificadores como short, long, signed, e unsigned podem ser usados com tipos inteiros para alterar o tamanho ou a faixa de valores (ex: long int, unsigned char).

Declaração e Inicialização de Variáveis

Uma variável é um espaço na memória do computador reservado para armazenar um valor. Antes de usar uma variável, você deve declará-la, especificando seu tipo e nome.

Declaração: tipo nome_da_variavel;

Inicialização: É o ato de atribuir um valor inicial a uma variável no momento da sua declaração ou posteriormente.


#include <stdio.h>

int main() {
    // Declaração de variáveis
    int idade;
    float salario;
    char inicial;
    double pi_valor; // Alterado de 'pi' para 'pi_valor' para evitar conflito com constante PI abaixo, se houver.

    // Inicialização de variáveis (atribuição de valores)
    idade = 30;
    salario = 2500.75f; // 'f' no final indica que é um float literal
    inicial = 'J';
    pi_valor = 3.1415926535;

    // Declaração e inicialização na mesma linha
    int ano = 2024;
    float altura = 1.75f;

    printf("Idade: %d anos\n", idade);
    printf("Salario: R$ %.2f\n", salario); // %.2f formata para 2 casas decimais
    printf("Inicial do nome: %c\n", inicial);
    printf("Valor de PI (double): %lf\n", pi_valor); // %lf para double
    printf("Ano: %d\n", ano);
    printf("Altura: %.2f m\n", altura);

    return 0;
}

Constantes (#define e const)

Constantes são valores que não podem ser alterados durante a execução do programa. Usar constantes torna o código mais legível e fácil de manter.

Usando #define (Diretiva de Pré-processador):

#define NOME_DA_CONSTANTE valor

O pré-processador substitui todas as ocorrências de NOME_DA_CONSTANTE pelo valor antes da compilação.


#include <stdio.h>

#define PI 3.14159f
#define MAX_ALUNOS 50

int main() {
    float raio = 5.0f;
    float circunferencia = 2 * PI * raio;

    printf("Circunferencia: %.2f\n", circunferencia);
    printf("Maximo de alunos: %d\n", MAX_ALUNOS);

    // PI = 3.14; // Erro! Não se pode atribuir valor a uma constante definida com #define
    return 0;
}

Usando a Palavra-chave const:

const tipo nome_da_constante = valor;

Cria uma variável cujo valor não pode ser modificado após a inicialização. É mais seguro e flexível que #define, pois respeita o escopo e o tipo.


#include <stdio.h>

int main() {
    const float G = 9.81f; // Aceleração da gravidade
    const int DIAS_SEMANA = 7;

    printf("Gravidade: %.2f m/s^2\n", G);
    printf("Dias na semana: %d\n", DIAS_SEMANA);

    // G = 10.0f; // Erro! Variável 'G' é constante.
    // DIAS_SEMANA = 8; // Erro!
    return 0;
}

Regras de Nomenclatura para Variáveis

  • Nomes de variáveis podem conter letras (maiúsculas e minúsculas), números e o caractere underscore (_).
  • O primeiro caractere deve ser uma letra ou underscore. Não pode ser um número.
  • C é case-sensitive, o que significa que idade, Idade e IDADE são três variáveis diferentes.
  • Não podem ser usadas palavras-chave reservadas da linguagem C (como int, float, if, while, etc.) como nomes de variáveis.
  • É uma boa prática usar nomes descritivos que indiquem o propósito da variável (ex: precoProduto, contadorDeAlunos).
  • Convenções comuns:
    • camelCase: minhaVariavelDetalhada (primeira letra minúscula, demais palavras com inicial maiúscula).
    • snake_case (ou underscore_case): minha_variavel_detalhada (palavras separadas por underscore, tudo minúsculo). Frequentemente usado em C.
    • PascalCase: MinhaVariavelDetalhada (todas as palavras com inicial maiúscula). Menos comum para variáveis em C, mais para tipos definidos pelo usuário.
    • Constantes: Frequentemente em MAIÚSCULAS com underscore (MINHA_CONSTANTE).

Escolha uma convenção e seja consistente no seu projeto.

Exemplos de Códigos (Atividades Resolvidas)

Atividade 1: Calculadora Simples (Soma)

Crie um programa que declare duas variáveis inteiras, atribua valores a elas, some esses valores e imprima o resultado.


#include <stdio.h>

int main() {
    int numero1;
    int numero2;
    int soma;

    numero1 = 15;
    numero2 = 27;

    soma = numero1 + numero2;

    printf("O primeiro numero e: %d\n", numero1);
    printf("O segundo numero e: %d\n", numero2);
    printf("A soma de %d + %d e: %d\n", numero1, numero2, soma);

    return 0;
}

Atividade 2: Conversão de Temperatura (Celsius para Fahrenheit)

Declare uma variável float para armazenar uma temperatura em Celsius. Converta-a para Fahrenheit usando a fórmula $F = (C \times \frac{9}{5}) + 32$ e imprima ambas as temperaturas.


#include <stdio.h>

#define FATOR_CONVERSAO (9.0f/5.0f) // Usar 9.0f e 5.0f para garantir divisão float
#define PONTO_CONGELAMENTO 32.0f

int main() {
    float celsius;
    float fahrenheit;

    celsius = 25.5f; // Exemplo de temperatura em Celsius

    fahrenheit = (celsius * FATOR_CONVERSAO) + PONTO_CONGELAMENTO;

    printf("%.1f graus Celsius equivalem a %.1f graus Fahrenheit.\n", celsius, fahrenheit);

    return 0;
}
