5. Estruturas Condicionais

Estruturas condicionais permitem que um programa tome decisões e execute diferentes blocos de código com base em certas condições serem verdadeiras ou falsas.

if, else if, else

A estrutura if executa um bloco de código se uma condição for verdadeira. O else if permite testar múltiplas condições em sequência, e o else executa um bloco se nenhuma das condições anteriores (if ou else if) for verdadeira.

Sintaxe:


if (condicao1) {
    // Bloco de código 1: executado se condicao1 for verdadeira
} else if (condicao2) {
    // Bloco de código 2: executado se condicao1 for falsa E condicao2 for verdadeira
} else if (condicaoN) {
    // Bloco de código N: ...
} else {
    // Bloco de código ELSE: executado se todas as condições anteriores forem falsas
}

Observações:

  • O bloco else if e o bloco else são opcionais.
  • Você pode ter múltiplos blocos else if.
  • Apenas um bloco de código (o primeiro cuja condição for verdadeira, ou o bloco else) será executado.

Exemplo: Classificação da Média Escolar


#include <stdio.h>

int main() {
    float media;
    printf("Digite a media do aluno: ");
    scanf("%f", &media);

    if (media >= 9.0f && media <= 10.0f) {
        printf("Conceito: A (Excelente)\n");
    } else if (media >= 7.0f && media < 9.0f) {
        printf("Conceito: B (Bom)\n");
    } else if (media >= 5.0f && media < 7.0f) {
        printf("Conceito: C (Regular - Recuperacao)\n");
    } else if (media >= 0.0f && media < 5.0f) {
        printf("Conceito: D (Insuficiente - Reprovado)\n");
    } else {
        printf("Media invalida!\n");
    }
    return 0;
}

switch e case

A estrutura switch é uma alternativa ao uso de múltiplos if-else if quando se deseja testar uma variável contra uma lista de valores constantes (inteiros ou caracteres).

Sintaxe:


switch (expressao_de_controle) {
    case valor1:
        // Bloco de código para valor1
        break; // Importante para sair do switch
    case valor2:
        // Bloco de código para valor2
        break;
    case valorN:
        // Bloco de código para valorN
        break;
    default: // Opcional
        // Bloco de código executado se nenhum case corresponder
        // Não precisa de break aqui, pois é o último
}
  • A expressao_de_controle deve resultar em um valor inteiro ou caractere.
  • Cada case define um valor constante a ser comparado.
  • O break é crucial. Sem ele, a execução "cai" para o próximo case (fall-through), o que raramente é desejado.
  • O default é opcional e é executado se nenhum case corresponder ao valor da expressão.

Exemplo: Dia da Semana


#include <stdio.h>

int main() {
    int dia;
    printf("Digite um numero de 1 a 7 para o dia da semana: ");
    scanf("%d", &dia);

    switch (dia) {
        case 1:
            printf("Domingo\n");
            break;
        case 2:
            printf("Segunda-feira\n");
            break;
        case 3:
            printf("Terca-feira\n");
            break;
        case 4:
            printf("Quarta-feira\n");
            break;
        case 5:
            printf("Quinta-feira\n");
            break;
        case 6:
            printf("Sexta-feira\n");
            break;
        case 7:
            printf("Sabado\n");
            break;
        default:
            printf("Numero invalido para dia da semana!\n");
    }
    return 0;
}

Uso do break e default

break no switch: Como visto, o break é usado para terminar a execução de um bloco case dentro de um switch e sair da estrutura switch. Se omitido, ocorre o "fall-through", onde o código do próximo case (e subsequentes, até encontrar um break ou o fim do switch) será executado, independentemente da condição do case.

default no switch: O rótulo default especifica um bloco de código a ser executado se nenhum dos valores dos rótulos case corresponder ao valor da expressão de controle do switch. É semelhante ao else final em uma cadeia if-else if-else. Seu uso é opcional, mas recomendado para tratar situações inesperadas.

Exemplos de Códigos (Atividades Resolvidas)

Atividade 1: Calculadora Simples com Switch

Crie uma calculadora que leia dois números e um operador (+, -, *, /). Use switch para realizar a operação.


#include <stdio.h>

int main() {
    float num1, num2, resultado;
    char operador;

    printf("Digite o primeiro numero: ");
    scanf("%f", &num1);
    printf("Digite o operador (+, -, *, /): ");
    scanf(" %c", &operador); // Espaço antes de %c para consumir newline anterior
    printf("Digite o segundo numero: ");
    scanf("%f", &num2);

    switch (operador) {
        case '+':
            resultado = num1 + num2;
            printf("%.2f + %.2f = %.2f\n", num1, num2, resultado);
            break;
        case '-':
            resultado = num1 - num2;
            printf("%.2f - %.2f = %.2f\n", num1, num2, resultado);
            break;
        case '*':
            resultado = num1 * num2;
            printf("%.2f * %.2f = %.2f\n", num1, num2, resultado);
            break;
        case '/':
            if (num2 != 0) {
                resultado = num1 / num2;
                printf("%.2f / %.2f = %.2f\n", num1, num2, resultado);
            } else {
                printf("Erro: Divisao por zero!\n");
            }
            break;
        default:
            printf("Operador invalido!\n");
    }
    return 0;
}

Atividade 2: Tipo de Triângulo

Leia três valores float que representam os lados de um triângulo. Verifique se eles podem formar um triângulo (a soma de quaisquer dois lados deve ser maior que o terceiro lado). Se formarem, classifique como:

  • Equilátero: todos os lados iguais.
  • Isósceles: dois lados iguais.
  • Escaleno: todos os lados diferentes.


#include <stdio.h>

int main() {
    float a, b, c;
    printf("Digite o lado a: "); scanf("%f", &a);
    printf("Digite o lado b: "); scanf("%f", &b);
    printf("Digite o lado c: "); scanf("%f", &c);

    // Verificar condição de existência de um triângulo
    if (a < b + c && b < a + c && c < a + b) {
        printf("Os lados podem formar um triangulo.\n");
        if (a == b && b == c) {
            printf("Tipo: Equilatero\n");
        } else if (a == b || a == c || b == c) {
            printf("Tipo: Isosceles\n");
        } else {
            printf("Tipo: Escaleno\n");
        }
    } else {
        printf("Os lados NAO podem formar um triangulo.\n");
    }
    return 0;
}
